# Cleaning Service Company Website

> **Project Status:** Initial setup complete. Business name and full content TBD.

A professional website for a cleaning service company. Built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack (Current)

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Linting:** ESLint

## Getting Started

```bash
# Install dependencies (if not already done)
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
